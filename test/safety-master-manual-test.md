# Safety Master 수동 테스트 체크리스트

## 테스트 환경
- 브라우저: Chrome (권장), Firefox, Safari
- 모바일: iOS Safari, Android Chrome
- 해상도: 데스크톱 (1920x1080), 태블릿 (768x1024), 모바일 (375x667)

## 1. 키보드 네비게이션 테스트

### 카드 모드
- [ ] ←/→ 화살표: 문제 이동 (이전/다음)
- [ ] ↑ 화살표: 힌트 표시
- [ ] ↓ 화살표: 정답 표시
- [ ] Enter: 답안 제출
- [ ] Shift+Enter: 줄바꿈 (답안 입력 중)
- [ ] Space: 카드 뒤집기
- [ ] V: 음성 입력 시작
- [ ] H: 힌트 표시
- [ ] A: 정답 표시
- [ ] M: 암기완료 표시
- [ ] W: 취약문제 표시

### 몰입 모드
- [ ] ←/→ 화살표: 문제 이동
- [ ] ↑ 화살표: 정답 표시
- [ ] ↓ 화살표: 힌트 표시
- [ ] Enter: 답안 제출
- [ ] V: 음성 입력 시작

### 슬라이드 모드
- [ ] ←/→ 화살표: 슬라이드 이동
- [ ] ↑/↓ 화살표: 정답 확인
- [ ] Space: 다음 슬라이드
- [ ] Enter: 슬라이드 뒤집기

## 2. 음성 입력 테스트

### 기본 음성 입력
- [ ] V키로 음성 입력 시작
- [ ] 마이크 권한 요청 처리
- [ ] 음성 인식 중 상태 표시 (🔴 중지 버튼)
- [ ] 음성 인식 결과 자동 삽입
- [ ] 음성 인식 완료 후 자동 제출 (옵션)

### 에러 처리
- [ ] 마이크 권한 거부 시 안내 메시지
- [ ] 음성 인식 실패 시 에러 메시지
- [ ] 브라우저 미지원 시 안내 메시지

## 3. 채점 시스템 테스트

### 완전 정답 시나리오

- [ ] 점수 80점 이상 (Excellent)
- [ ] 피드백 메시지 표시
- [ ] 키워드 매칭 결과 표시

### 부분 정답 시나리오
- [ ] 핵심 키워드 일부 포함 답안
- [ ] 점수 40-79점 (Good/Fair)
- [ ] 매칭된/누락된 키워드 표시

### 오답 시나리오
